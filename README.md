# PyUsbLamp
Mailbox Friends Alert (Dream Cheeky) Driver in Python.

## The Device
Basically, it is a RGB LED connected with USB. 

**Vendor ID:** `0x1d34`  
**Product ID:** `0x000a`

![UsbLamp](UsbLamp.png)

## Dependencies
pyusb

## Installation
1. Install from Pypi

   `pip install pyusblamp`

or

2. Install from GitHub

   `pip install git+https://github.com/onelife/PyUsbLamp.git`
